Zepto SQL Data Analysis Project
📌 Project Overview
This project analyzes a Zepto product dataset using SQL to explore product pricing, discounts, inventory, and category performance.
The goal of this project is to demonstrate SQL skills used in real-world data analysis, including:
1.Data exploration
2.Data cleaning
3.Business insights
4.Product pricing analysis
5.Inventory analysis

This project simulates how a Data Analyst would analyze product data for an e-commerce or quick-commerce company like Zepto.

🗂 Dataset Information

📂 Dataset
The dataset contains product-level information such as:
1.Product Name
2.Category
3.MRP
4.Discount Percentage
5.Discounted Selling Price
6.Available Quantity
7.Product Weight
8.Stock Availability

This project demonstrates the following SQL concepts:
1.Table creation
2.Data exploration
3.Data cleaning
4.Filtering and sorting
5.Aggregate functions
6.GROUP BY analysis

📊 Analysis Performed
1.Some key analyses in this project:
2.Identified top discounted products
3.Found high-price products that are out of stock
4.Calculated estimated revenue by category
5.Determined categories with highest average discounts
6.Calculated price per gram to find best-value products
7.Categorized products by weight (Low / Medium / Bulk)
